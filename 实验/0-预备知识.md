# 预备知识

## 实验环境

教程演示了如何使用P4，Stratum和ONOS构建基于IPv6的叶脊数据中心结构，练习将集中在以下内容上：

* Stratum API（P4Runtime，gNMI，gNOI）的使用
* 将ONOS与通过任意P4程序编程的网络设备一起使用
* 编写ONOS应用以提供控制逻辑（桥接，路由，ECMP等）
* 测试Mininet中的bmv2
* 基于PTF的P4单元测试

我们直接采用提供的虚拟机作为实验环境来进行练习。
